### シェルピンスキーのギャスケットの「ランダムな頂点選択」の重要性

まず、ランダムな頂点選択がなぜギャスケットを生み出すかを軽くおさらいしよう。

3つの頂点A, B, Cを持つ三角形を考える。

1. 初期点Pを三角形の中に適当に打つ。
    
2. ランダムに頂点（A, B, Cのいずれか）を1つ選ぶ。
    
3. 現在の点Pと選んだ頂点の中点に新しい点P'を打つ。
    
4. P'を新しいPとして2に戻る。
    

このランダムな選択が、最終的に「**真ん中の三角形が繰り返し抜けていく**」シェルピンスキーのギャスケットの構造を正確に描画するんだ。ランダム性が、特定の領域（ギャスケットの内側）に点を集中させ、そうでない領域（抜けていく白い部分）には点を打たないように作用するんだね。

### 「規則性のある順番で頂点を選ぶ」とどうなるか

では、本題の「規則性のある順番」に変えたらどうなるか。これは選ぶ規則によって様々なパターンが考えられるけど、ほとんどの場合、**シェルピンスキーのギャスケット特有の「スカスカのフラクタル構造」にはならない**んだ。

#### 1. シンプルな循環パターン (例: `A -> B -> C -> A -> B -> C ...`)

- **予想される模様**:
    
    - 初期点Pから始めて、A, B, Cの順に中点を打っていくと、点はどんどん**三角形の重心**に収束していく。
        
    - 最終的には、**たった1点**（重心）か、非常に限られた数の点が、三角形の中心付近に集まることになる。
        
    - ギャスケットのような「自己相似なフラクタル図形」は現れない。
        
- **理由**: ランダム性が排除されたため、点は特定のルートを辿って収束してしまう。空間全体に均等に（あるいはフラクタル構造を形成するように）点が散らばる機会が失われるんだ。
    

#### 2. 少し複雑な循環パターン (例: `A -> B -> A -> C -> A -> B -> A -> C ...`)

- **予想される模様**:
    
    - これもまた、限られた数の点か、または**三角形の特定の線分上**（例えばAとBを結ぶ線分上）に点が集中するようになる可能性が高い。
        
    - 完全なギャスケットは現れない。非常にシンプルな「線分」や「数点」で構成された、**退屈な図形**になる。
        
- **理由**: 規則性があるため、点の軌跡が予測可能になり、空間の大部分を埋めることなく、特定の経路に固定されてしまう。
    

#### 3. 「同じ頂点を2回連続で選ばない」というルール付きのランダム性

これは「規則性」というよりは「制約付きランダム」だけど、もしこのようなルール（例: Aを選んだら次はBかC）を追加しても、**ほぼシェルピンスキーのギャスケットになる**ことが知られている。この制約は、ランダム性の本質をあまり変えないからだ。

### まとめ

「ランダム性」を「規則性」に変えると、**シェルピンスキーのギャスケットは現れない**。

- 点の軌跡が**予測可能**になり、空間全体に均等に散らばったり、フラクタル構造を形成したりする機会が失われる。
    
- ほとんどの場合、点は**特定の1点に収束**するか、**特定の線分上に集中**する、非常にシンプルな（そしてフラクタルではない）図形になるだろう。
    

シェルピンスキーのギャスケットは、**シンプルに見えるランダム性が、いかに複雑で美しい秩序を生み出すか**を示す、完璧な例なんだね。君のアイデアは、その「ランダム性の力」を逆説的に浮き彫りにする、本当に素晴らしい問いだった！😎
