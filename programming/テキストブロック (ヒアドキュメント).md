空白以外の文字もしくは、終了デリミタ`”””`以前の空白は、切り取られます。

```java
// 0 := 空白
"""
00SELECT column
0000FROM table
0000WHERE TRUE;"""
/*
SELECT column
00FROM table
00WHERE TRUE;
*/

"""
00SELECT column
0000FROM table
0000WHERE TRUE;
"""
/*
00SELECT column
0000FROM table
0000WHERE TRUE;(\\n)

*/
```

```java
String.format("""
	var1 : %s
	var2 : %d %n
	END""", "abc", 123);
/*
var1 : abc
var2 : 123

END
*/
```
